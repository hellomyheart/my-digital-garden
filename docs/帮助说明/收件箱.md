# 收件箱

- 在这里可以写一些杂乱的笔记，方便之后分类。
- 可以使用无格式的文本，也可以使用md格式。
- 核心是写下心中所想（灵感笔记），而不是过多的思考，忘记了最初的思想。
- 在这里也可以使用双链。
  - 比如这样 [[待办事项]]
- 对于没有意义的，陈旧的笔记，可以适当删除。
- git记录里会保留，不必害怕。
